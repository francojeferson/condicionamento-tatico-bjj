executar 1 round

| exercicios                         | repetiçoes |
| ---------------------------------- | ---------- |
| rotaçao de ombro lateral           | 30s+30s    |
| passada alternada com joelho       | 1min       |
| ponte alternada lateral            | 1min       |
| troca de base com rotaçao toracica | 2min       |
| ponte 90/90                        | 1min       |
